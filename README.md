# JDBDT 

JDBDT (Java Database Delta Testing) is a library for 
Java database application testing. Visit [http://jdbdt.org](http://jdbdt.org)
for more information.

# Compilation requirements

* Maven 3.2 or higher
* Java 8 or higher
